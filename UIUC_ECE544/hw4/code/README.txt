# UIUC ECE544NA Assignment 4

########################################################################################
Instruction for running the python3 code

All the required codes are in the folder named code.

    RBM.py contains the defined functions and classes for Code_From_Scratch part
    
    tf_RBM.py contains the defined functions and classes for TensorFlow part

    Code_From_Scratch.py is the main part for part 2
    
    TensorFlow.py is the main part for part 3

########################################################################################
To run the code (in Ubuntu 16.04 with Anaconda Python)

    1. $: source activate tensorflow

    2. $: python Code_From_Scratch.py
    
    3. $: python TensorFlow.py
    
**Note**
    The result will print out on the terminal and/or save to local folder named ./result/
