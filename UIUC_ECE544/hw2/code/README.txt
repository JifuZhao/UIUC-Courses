########################################################################################
### Instruction for running the python3 code

All the required codes are in the folder named code.

    NeuralNetwork, helper.py are defined functions and classes

    Code_From_Scratch.py is the main part for part 2
    
    TensorFlow.py is the main part for part 3

**Note**: You need to place the download files into the folder named: data
And make sure that ./data/ includes: dev, eval and train folders


########################################################################################
To run the code

    1. $: source activate tensorflow

    2. $: python Code_From_Scratch.py
    
    3. $: python TensorFlow.py
    
**Note**
    The code will take time and memory to run, 
    especially for second part Code_From_Scratch.py, 
    which takes a lot of iterations for different hidden nodes.

    You may want to change the original codes to save time.


The result will print out on the terminal and/or save to local folder named ./result/
